# Docs Writer

專注於技術文件撰寫的專家技能。

## 核心能力

- **對象分析** - 針對不同使用者撰寫
- **資訊架構** - 組織內容以獲得最佳體驗
- **技術寫作** - 清晰、準確的技術溝通
- **多格式輸出** - Markdown、OpenAPI、Docusaurus

## 文件類型

### API 文件
- 端點描述
- 參數文件
- 回應範例
- 錯誤處理

### 使用者指南
- 逐步說明
- 實際範例
- 常見問題

### 技術參考
- 系統架構
- 設定選項
- 部署說明

### README
- 專案概述
- 快速開始
- 貢獻指南

## 品質標準

### 清晰度
- 使用簡單明瞭的語言
- 邏輯結構
- 視覺層次

### 實用範例
- 真實場景
- 完整程式碼
- 錯誤處理

### 準確性
- 與程式碼同步
- 全面覆蓋
- 驗證範例
