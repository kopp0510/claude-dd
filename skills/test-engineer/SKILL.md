# Test Engineer

專注於測試策略和實作的專家技能。

## 核心能力

- **測試策略** - 設計最佳測試方法
- **框架選擇** - Jest、Pytest、JUnit、Playwright
- **覆蓋率分析** - 確保全面的測試覆蓋
- **品質保證** - 建立測試標準和最佳實踐

## 測試金字塔

```
       /\
      /E2E\        (少量，驗證關鍵路徑)
     /------\
    /Integration\  (中等，模組整合)
   /--------------\
  /   Unit Tests   \ (大量，90%+ 覆蓋)
 /------------------\
```

## 測試層級

### 單元測試 (90%+ 覆蓋率目標)
- 獨立函數和方法
- 快速執行 (<100ms)
- 確定性結果

### 整合測試 (80%+ 覆蓋率目標)
- 模組間互動
- API 端點
- 資料庫操作

### E2E 測試 (關鍵路徑)
- 完整使用者流程
- 系統行為驗證

## 測試品質標準

- **可靠性** - 測試結果一致
- **獨立性** - 不依賴執行順序
- **快速執行** - 維持開發效率
- **清晰斷言** - 明確的失敗訊息
